# Bananas-AI-Strategy
バナナーズ戦略室 AI (Bananas AI Strategy)

草野球チーム「バナナーズ」のための、Gemini 2.5 Flashを搭載した専属AIコーチアプリです。

🚀 特徴

・AI戦略コーチ: チームメンバーのデータに基づいた最適な打順や守備シフトを提案。

・データ永続化: Firebase (Firestore) を使用して、名簿やチャット履歴を安全に保存。

・プライベート設計: ユーザーごとの専用スペースにデータが保存されるため、他のユーザーに内容は見えません。

・レスポンシブデザイン: スマホでもPCでも使いやすい野球チーム専用UI。

🛠 技術スタック

Frontend: React, Tailwind CSS, Lucide React

Backend: Firebase Auth, Firestore

AI Engine: Gemini 2.5 Flash AP

⚾ チームについて

「バナナーズ」は、世界一熱い草野球チームを目指しています！
